# 뉴스 기반 챗봇

구글 뉴스를 검색하고, 제미나이 API를 사용하여 뉴스를 요약하고 대화할 수 있는 웹 애플리케이션입니다.

## 기능

- 🔍 키워드로 구글 뉴스 검색 (5개)
- 📝 제미나이 API를 사용한 뉴스 자동 요약
- 💬 뉴스 기반 대화형 챗봇

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
python app.py
```

3. 브라우저에서 접속:
```
http://127.0.0.1:5000
```

## 사용 방법

1. 검색창에 키워드를 입력하고 "검색" 버튼을 클릭합니다.
2. 검색된 뉴스 5개가 표시됩니다.
3. 뉴스 요약이 자동으로 생성됩니다.
4. 채팅창에서 뉴스에 대해 질문할 수 있습니다.

## 기술 스택

- **Backend**: Flask
- **AI**: Google Gemini API
- **Frontend**: HTML, CSS, JavaScript
- **뉴스 검색**: Google News RSS

## 주의사항

- 이 애플리케이션은 로컬 환경에서만 실행됩니다.
- 제미나이 API 키가 코드에 포함되어 있습니다. 프로덕션 환경에서는 환경 변수로 관리하는 것을 권장합니다.
